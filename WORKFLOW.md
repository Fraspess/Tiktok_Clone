# Як працювати з задачами

## 1. Знайди свої задачі

1. Заходиш на GitHub репозиторій
2. Натискаєш вкладку **Issues**
3. Натискаєш **Filters** і вводиш:
```
assignee:@me
```
4. Бачиш тільки свої задачі

---

## 2. Дошка (Projects)

1. Натискаєш вкладку **Projects**
2. Заходиш в дошку проєкту
3. Вгорі натискаєш **Filters** і вводиш:
```
assignee:@me milestone:v1-Core
```
> Міняй `v1-Core` на `v2-Social` або `v3-Real-time-Admin` залежно від етапу
4. Тепер бачиш тільки свої картки

---

## 3. Починаєш задачу

1. Знаходиш свою картку в колонці **Todo**
2. Перетягуєш її в **In Progress**
3. Робиш задачу

---

## 4. Готово

1. Перетягуєш картку в **Done**
2. Закриваєш issue (кнопка **Close issue** всередині)

---

## 5. З чого починати

Спочатку беріть задачі з **Milestone v1 — Core**.

Перевірити milestone можна так:
1. Вкладка **Issues**
2. Натискаєш **Milestone** → вибираєш `v1 — Core`
3. Бачиш всі задачі першого етапу

> ⚠️ Не беріть задачі з v2 або v3 поки не закрито v1
